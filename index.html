Projeto 1: Corretor de Provas

Uma aplicação móvel multiplataforma, desenvolvida em Flutter, que automatiza a correção de provas de múltipla escolha. Utilizando a câmera do dispositivo, o sistema aplica técnicas de visão computacional + inteligência artificial com um backend em Python para analisar as folhas de resposta, comparar com um gabarito mestre e fornecer o resultado em um curto espaço de tempo. O projeto inclui gestão de turmas e provas, armazenamento persistente de dados e a capacidade de exportar relatórios detalhados para ficheiros Excel (.xlsx). A arquitetura cliente-servidor garante que o processamento pesado seja feito na nuvem, mantendo o aplicativo leve e rápido.

Projeto 2: Previsão Fórmula 1

Este projeto de ciência de dados desenvolve e avalia modelos de machine learning para simular o grid de largada de futuras corridas de Fórmula 1. O sistema utiliza dados históricos oficiais para treinar modelos preditivos e aplica fatores de desempenho calculados para refinar os resultados, gerando uma classificação simulada que leva em conta a performance histórica de equipes e pilotos. O projeto compara a performance de um modelo de base (Regressão Linear) com um modelo mais avançado (Random Forest) e implementa uma lógica híbrida para tratar pilotos novatos, utilizando dados de suas temporadas de estreia para criar fatores de desempenho relevantes.

Projeto 3: Art Seeker

O Art Seeker é uma aplicação de Inteligência Artificial voltada para a busca interativa em grandes acervos de obras de arte. A solução utiliza modelos de NLP (Processamento de Linguagem Natural) para compreender consultas em linguagem natural, como “pinturas renascentistas com anjos e temas religiosos”.O sistema transforma essas consultas em vetores semânticos (embeddings) e realiza a comparação com descrições das obras no acervo.Além disso, um CrossEncoder baseado em redes neurais reavalia os pares [consulta, obra], garantindo maior precisão na ordenação dos resultados.

Projeto 5: Previsão Mundial RF

Este projeto é um simulador que utiliza a Simulação de Monte Carlo para estimar a probabilidade de cada time vencer o Mundial de Clubes 2025, executando o torneio 1000 vezes. A previsão de cada partida individual é realizada por um algoritmo de Machine Learning, o Random Forest, que calcula as chances de vitória, empate ou derrota com base em estatísticas detalhadas. O resultado final consolida essas milhares de simulações para determinar os favoritos mais prováveis.
